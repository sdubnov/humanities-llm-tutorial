# Lesson 0 — First Contact with LLMs

## Goal

Learn the difference between casual chat, structured prompting, and the beginning of a reproducible research workflow.

## Part A — Open a browser-based LLM

Start with one of these:

- ChatGPT: good general-purpose assistant and document analysis interface
- Claude: strong for long documents, close reading, and prose explanation
- Gemini: useful for Google ecosystem users and multimodal work
- OpenAI Playground: useful for testing system prompts, temperature, and model settings
- Hugging Face Spaces: useful for trying open-source models and community demos

## Part B — Hello-world prompt

Open one LLM in your browser and paste:

```text
Explain the difference between metaphor, simile, metonymy, and irony in simple terms. Give one example of each.
```

Now ask:

```text
Turn the answer into a reusable checklist for analyzing a poem or song lyric.
```

Now ask:

```text
Apply the checklist to this line:
"The city was a broken mirror, reflecting every dream I had lost."
Return the answer as a table.
```

## What changed?

You moved through three stages:

1. Chat: asking for explanation.
2. Prompting: asking for a specific reusable format.
3. Research method: applying the same format to a new text.
