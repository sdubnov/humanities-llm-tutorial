# Lesson 6 — Mini Research Project

## Goal

Complete a small reproducible LLM-assisted humanities analysis project.

## Project steps

### Step 1 — Choose a research question

Example:

```text
How does a group of texts use metaphor to construct identity, conflict, memory, or place?
```

### Step 2 — Choose a small corpus

Start small:

- 3 poems
- 3 speeches
- 3 song lyrics
- 3 excerpts from a novel
- 3 essays

### Step 3 — Choose one theory text

Use one article, book chapter, or lecture note as the theoretical basis.

### Step 4 — Build a taxonomy

Use the prompt from Lesson 4.

### Step 5 — Analyze the corpus

Apply the same prompt to each text.

### Step 6 — Evaluate outputs

Ask:

- Did the model follow the taxonomy?
- Did it quote evidence?
- Did it overinterpret?
- Did it miss important patterns?
- Were categories applied consistently?

### Step 7 — Revise the method

Update the taxonomy and prompt.

### Step 8 — Write a short report

Include:

1. research question
2. corpus
3. theory source
4. taxonomy
5. prompt protocol
6. sample analyses
7. evaluation
8. limitations
9. next steps

## Final deliverables

```text
projects/my_project/
├── research_question.md
├── corpus_notes.md
├── taxonomy_v1.md
├── prompt_protocol.md
├── analyses/
├── evaluation.md
└── final_report.md
```

## Key idea

The goal is not to prove the LLM is correct.

The goal is to build a transparent, testable, revisable research workflow.
