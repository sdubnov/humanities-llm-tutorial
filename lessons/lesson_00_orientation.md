# Lesson 0 — From Chatting to Research Workflows

## Goal

Understand the difference between casually chatting with an LLM and using an LLM as part of a structured research method.

## The basic progression

```text
chatting → prompting → structured prompting → file-based workflow → agents → research system
```

## Chatting

Casual chatting is useful for brainstorming, but it is not yet a research method.

Example:

```text
What is metaphor?
```

This can be useful, but it is difficult to reproduce or evaluate.

## Prompting

Prompting means giving the model a more explicit task.

Example:

```text
Define metaphor, give three examples, and explain how to identify it in a poem.
```

## Structured prompting

Structured prompting turns the prompt into a protocol.

Example:

```text
Analyze the passage using these fields:

1. quoted passage
2. literary device
3. definition
4. evidence
5. possible alternative reading
6. confidence
```

## Research workflow

A research workflow saves the prompt, input text, output, and evaluation.

That means the work can be repeated, revised, and compared.

## Key idea

In this course, prompts are not just questions.

They are part of the research method.
