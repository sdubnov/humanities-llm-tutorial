# Lesson 1 — Environment and Repository Setup

## Goal

Set up a simple file-based environment for LLM research.

## Recommended tools

You need:

- a web browser
- a text editor such as Zed, VSCode, or Obsidian
- a GitHub repository
- access to an LLM such as ChatGPT, Claude, Gemini, or Hugging Face Spaces

## Optional exploration environments

Before using the repository workflow, students can experiment in:

- OpenAI Playground
- Hugging Face Spaces
- ChatGPT
- Claude
- Gemini

These are good for early exploration.

The repository is used once the work becomes more systematic.

## Suggested folder structure

```text
humanities-llm-tutorial/
├── lessons/
├── prompts/
├── templates/
├── examples/
├── projects/
├── session_notes/
└── advanced_topics/
```

## Why files matter

Files make the work:

- reproducible
- shareable
- editable
- comparable
- citable as method

## Basic rule

If a prompt, rule, taxonomy, or output matters, save it as a file.
