# Final Mini Project

## Goal

Build a small reproducible LLM-assisted humanities analysis project.

## Deliverables

1. One theory excerpt or article
2. One small corpus of 3–5 short texts
3. A taxonomy built from the theory
4. A structured prompt
5. A multi-turn session
6. A human critique
7. A revised taxonomy
8. A final reflection

## Folder checklist

```text
theory_texts/
source_texts/
prompts/
rules_methods/
outputs/
evaluations/
session_notes/
```

## Final reflection questions

- What did the LLM do well?
- Where did it over-interpret?
- Which categories were most stable?
- Which categories needed human judgment?
- Did multi-turn interaction improve the method?
- What would be needed for a larger study?
