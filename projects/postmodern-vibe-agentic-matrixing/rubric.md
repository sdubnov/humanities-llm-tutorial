# Rubric: Postmodern Vibe Agentic Matrixing

| Category | Excellent | Proficient | Developing |
|---|---|---|---|
| Agent design | Agent instructions are precise, theoretically meaningful, and strongly bounded. | Agent instructions are clear but rely on some generic terms. | Agents are vague, overlapping, or produce general summaries. |
| Theory-to-metric translation | Abstract concepts are operationalized carefully without pretending to be objective. | Concepts are translated into usable scores with some oversimplification. | Scores are used without explaining what they mean. |
| Matrix quality | Matrix is complete, consistent, and includes justifications and reliability warnings. | Matrix is mostly complete but some justifications are thin. | Matrix has missing scores, weak evidence, or no reliability warnings. |
| Use of script/report | Script is run successfully and the report is interpreted critically. | Script is run, but interpretation is mostly descriptive. | Script is not run or the report is accepted without critique. |
| Methodological critique | Reflection identifies summary bias, reputation bias, false precision, and agent drift with examples. | Reflection identifies some limitations but lacks specificity. | Reflection treats LLM scores as authoritative. |
| Humanities interpretation | Student connects numerical patterns back to meaningful literary questions. | Student makes some interpretive connections. | Student leaves the matrix disconnected from literary interpretation. |
