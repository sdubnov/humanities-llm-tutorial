# Sources to Consult

These links are starting points for the instructor and students.

## Multi-turn LLM interaction

- NeurIPS 2025 Workshop on Multi-Turn Interactions in Large Language Models
- OpenReview page for MTI-LLM @ NeurIPS 2025

## CLI and editor-based LLM agents

- OpenAI Codex CLI GitHub repository
- Anthropic Claude Code documentation/product page
- Google Gemini CLI documentation
- Gemini CLI codelab

## Why these matter

The course does not require students to build full software agents. But these sources show the broader movement from single-turn prompting toward multi-turn, tool-using, file-aware, and agentic LLM systems.
