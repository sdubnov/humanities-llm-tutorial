# Advanced Capstone — Humanities Research Platform Prototype

## Goal

Build a small prototype that combines:

- one skill
- one theory source
- one small corpus
- manual RAG
- one loop pattern
- human evaluation
- optional fine-tuning data preparation

## Deliverables

1. `skills/my_skill/SKILL.md`
2. `examples/`
3. `rubric.md`
4. `output_schema.json`
5. `rag_sources.md`
6. `loop_protocol.md`
7. `evaluation_harness/`
8. `annotations.jsonl`
9. `final_report.md`

## Reflection questions

- What did the skill make reusable?
- Which tools would improve the workflow?
- What information should be retrieved?
- Which loop pattern helped most?
- What should remain human-controlled?
- Would fine-tuning be justified?
