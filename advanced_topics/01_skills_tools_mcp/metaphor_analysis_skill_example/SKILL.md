# Metaphor Analysis Skill

Use this skill to analyze metaphor in a short literary, historical, biblical, rhetorical, or lyric text.

For each metaphor, identify:

1. exact evidence
2. device type
3. source domain
4. target domain
5. affect or tone
6. interpretation
7. alternative reading
8. confidence from 1–5

Do not make cultural, historical, political, or theological claims unless the text or provided context supports them.

Distinguish:

- evidence
- interpretive move
- possible meaning
- uncertainty

## Method

1. Read the passage carefully.
2. Identify possible figurative language.
3. Decide whether the device is metaphor, simile, symbol, motif, or another device.
4. Quote exact evidence.
5. Identify the source domain.
6. Identify the target domain.
7. Describe the affect or tone.
8. Explain the interpretation.
9. Add an alternative reading.
10. Assign a confidence score from 1 to 5.
