# Allowed Sources

The model may use only the following sources unless the instructor explicitly provides more context.

## Test Texts

```text
test_texts/
```

## Guides

```text
guides/metaphor_taxonomy.md
guides/uncertainty_policy.md
guides/citation_policy.md
```

## Prior Context

Prior annotations may be used only if they are pasted into the prompt or included in the context packet.

## Disallowed Sources

The model should not use:

- outside biography
- unsupported cultural assumptions
- invented historical context
- uncited web knowledge
- unstated theory sources
- personal speculation presented as evidence
