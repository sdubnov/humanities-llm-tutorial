# Advanced Lesson 4 — Evaluation Harnesses

## Goal

Build repeatable tests for prompts, models, taxonomies, and workflows.

## What is a harness?

```text
same texts
same prompt
same taxonomy
different models or prompt versions
human evaluation
comparison report
```

## Example folder

```text
evaluation_harness/
├── test_texts/
├── prompts/
├── taxonomies/
├── model_outputs/
├── human_scores.csv
└── report.md
```

## Suggested scoring categories

- instruction following
- evidence accuracy
- category consistency
- ambiguity handling
- overinterpretation
- cultural claim support
- usefulness for research

## Lab

1. Choose 3 short texts.
2. Use 1 taxonomy.
3. Run 2 prompt versions.
4. Score each output using the rubric.
5. Write a short comparison report.
