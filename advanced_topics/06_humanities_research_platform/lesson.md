# Advanced Lesson 6 — Building a Humanities Research Platform

## Goal

Design a small humanities research platform that combines skills, tools, retrieval, loops, and evaluation.

## Platform architecture

```text
Corpus
→ Theory library
→ Skills
→ RAG
→ Agents
→ Loops
→ Evaluation harness
→ Reports
```

## Final advanced capstone

Build a small platform prototype:

1. Define one humanities skill.
2. Add a small corpus.
3. Add one theory source.
4. Use manual RAG.
5. Run an Analyst–Skeptic–Editor loop.
6. Score outputs with a rubric.
7. Save results.
8. Prepare 5 examples for possible fine-tuning.
