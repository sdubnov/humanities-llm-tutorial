# Introductory Course Instructional Videos

## Purpose

This page collects introductory instructional videos that support the first course:

```text
First contact with LLMs
→ prompting as method
→ files and memory
→ humanities concepts and taxonomies
→ multi-turn interaction
→ reasoning traces
→ role-based agents
→ reproducible humanities workflows
```

The introductory course should use videos selectively.

The goal is not to make students watch many technical tutorials. The goal is to give them a few accessible entry points into LLM use, prompting, humanities applications, and multi-turn interaction.

# Lesson 3 — Sessions, CLI Tools, Memory, and Commands

## Suggested Video Use

There may not be one ideal beginner video for this lesson, because the lesson is about the course's own workflow:

```text
session memory
externalized notes
markdown summaries
restartability
CLI or file-based workflows
```

Recommended approach:

Use short in-class demonstrations instead of assigning a video.

Possible instructor demo:

```text
1. Ask an LLM to analyze a text.
2. Close or restart the session.
3. Restore the work from a markdown session summary.
4. Compare what is preserved and what is lost.
```

Teaching note:

This lesson is best taught through live workflow demonstration.

# Lesson 6 — Reasoning Traces: CoT, ToT, and PoT

## Suggested Video Use

For the introductory course, do not overemphasize technical CoT / ToT videos.

Instead, teach reasoning traces through class examples.

Recommended framing:

```text
CoT = visible evidence-to-claim reasoning
ToT = competing interpretive branches
PoT = procedural version of a method
```

If using videos, assign them only as optional background.

---

# Lesson 7 — Agent Roles for Interpretive Analysis

## Suggested Video Use

For the introductory course, role-based agents should be taught with course examples rather than technical agent-framework videos.

Recommended in-class activity:

```text
Run the same passage through:
Analyst
Skeptic
Editor
Archivist
```

Then ask students:

```text
Which role improved the method?
Which role only added style?
Which role preserved reproducibility?
```

- avoid embedding them deeply inside lesson files
- review links before teaching each offering
- replace broken links with updated searches or official channels
